# 🏢 Employee Attrition Prediction

## 📌 Project Overview

This is an End-to-End Machine Learning project that predicts whether an employee is likely to leave the company based on various employee-related factors.

The project includes:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Hyperparameter Tuning
- Streamlit Web Application
- Machine Learning Model Deployment

---

## 📊 Dataset

Dataset: IBM HR Analytics Employee Attrition Dataset

Target Variable:

- Attrition
  - 0 = Employee Stays
  - 1 = Employee Leaves
